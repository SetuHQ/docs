## Unpausing a paused mandate

<Text size="small" weight="600">
  PREREQUISITES&nbsp;
</Text>
<a href="/payments/umap/mandates/generic" target="_self">
  ðŸŒŸ UPI mandates generic operations â†—

<hr class="primary" />

Only the end customer has the option of unpausing a paused UPI mandate from their UPI app and there is no merchant intervention required.

<p>
  When a customer initiates mandate pause from their UPI app, UPI Setu will send
  notifications to the merchant for following events:
</p>
<ul>
  <li>
    <b>mandate_operation.unpause.initiated</b>
  </li>
  <li>
    <b>mandate.live</b> (if mandate unpause is successful)
  </li>
  <li>
    <b>mandate_operation.unpause.success</b> (if mandate unpause is successful)
  </li>
  <li>
    <b>mandate.paused</b> (if mandate unpause fails)
  </li>
  <li>
    <b>mandate_operation.unpause.failed</b> (if mandate unpause fails)
  </li>
</ul>
<p>
  Read more about mandate status notifications{" "}
  <a href="/payments/umap/notifications/mandates">here</a> and mandate unpause
  operation notifications{" "}
  <a href="/payments/umap/notifications/mandates/unpause">here</a>
</p>
<p>The notifications are depicted in the below diagram</p>
<img
  src="https://storage.googleapis.com/strapi-assets/latest/update-mandate-customer-unpause/update-mandate-customer-unpause.png"

<p>
  After receiving <b>mandate_operation.unpause.initiated</b> notification, the
  merchant can also use{" "}
  <Link href="/payments/umap/mandates/generic/unpause#check-mandate-operation-status">
    check mandate operation status API
  </Link>{" "}
  to check if mandate unpause is <b>successful</b>
</p>

#### Check mandate operation status (optional)

After receiving pause operation initiated notification, the merchant can use this API to check the status of a mandate unpause operation.

<table>
  <tbody>
    <tr>
      <th>Staging host</th>
      <td>
        <code>https://umap.setu.co/api</code>
      </td>
    </tr>
    <tr>
      <th>Method</th>
      <td>
        <code>GET</code>
      </td>
    </tr>
    <tr>
      <th>Path</th>
      <td>
        <code>/v1/merchants/mandate-operations/&#123;id&#125;</code>
      </td>
    </tr>

        <code>merchantId</code>
