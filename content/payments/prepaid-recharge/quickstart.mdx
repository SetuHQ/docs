## Overview

The **Mobile Prepaid Recharge API** enables partners to integrate and automate mobile recharge functionality. This API allows authorized partners to perform prepaid mobile recharges across different operators, track recharge status, and manage transactions. It also supports querying recharge details for various filters, including partner-specific configurations.

### Key Features:
- **Mobile Prepaid Recharge**: Seamlessly perform prepaid mobile recharges for different operators.
- **Recharge Status Tracking**: Check the status of recharges in real-time.
- **Flexible Configurations**: Support for multiple partners with configurable access, headers, and request transformation.


### Authentication:
The API uses authentication mechanisms that allow access only to authorized partners. Each partner is provided with:
- **Access Keys**: Specific to each partner for secure authentication.


### Rate Limits:
To ensure fair usage, the API imposes rate limits on the number of requests a partner can make. Exceeding the rate limit will result in `429 Too Many Requests` responses.
