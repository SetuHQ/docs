## API reference

This document provides detailed API specifications for Bundled Bank Account Verification.

<hr class="primary" />

### Create Bundled BAV request

Create a verification request to generate a URL for the bundled BAV flow.

<table>
	<tbody>
		<tr>
			<th>Method</th>
			<td>
				<code>POST</code>
			</td>
		</tr>
		<tr>
			<th>Path</th>
			<td>
				<code>/api/bundled-bav</code>
			</td>
		</tr>
		<tr>
			<th>Sandbox base URL</th>
			<td>
				<code>https://dg-sandbox.setu.co</code>
			</td>
		</tr>
		<tr>
			<th>Production base URL</th>
			<td>
				<code>https://dg.setu.co</code>
			</td>
		</tr>
	</tbody>
</table>

### Webhook notification

After the customer completes the verification, Setu sends a webhook notification to your configured endpoint.

<table>
	<tbody>
		<tr>
			<th>Method</th>
			<td>
				<code>POST</code>
			</td>
		</tr>
		<tr>
			<th>Endpoint</th>
			<td>Your configured webhook URL</td>
		</tr>
	</tbody>
</table>

<hr class="tertiary" />

#### Webhook payload

<table>
	<thead>
		<tr>
			<th>Parameter</th>
			<th>Type</th>
			<th>Description</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>
				<code>requestId</code>
			</td>
			<td>String (UUID)</td>
			<td>Unique identifier for this verification request</td>
		</tr>
		<tr>
			<td>
				<code>data</code>
			</td>
			<td>Object</td>
			<td>Bank account details (see below)</td>

				<code>bavMethod</code>
			</td>
			<td>String (Enum)</td>
			<td>
				BAV method used for verification. Values:{" "}
				<code>PENNY_DROP</code>, <code>REVERSE_PENNY_DROP</code>,{" "}
				<code>MOBILE_TO_BANK</code> or <code>VPA_TO_BANK</code>
			</td>
		</tr>
		<tr>
			<td>
				<code>success</code>
			</td>
			<td>Boolean</td>
			<td>Indicates if verification was successful</td>
		</tr>
		<tr>
			<td>
				<code>message</code>
			</td>
			<td>String</td>
			<td>Descriptive message about the verification result</td>
		</tr>
		<tr>
			<td>
				<code>timestamp</code>
			</td>
			<td>String</td>
			<td>UTC timestamp of verification</td>
		</tr>
	</tbody>
</table>
