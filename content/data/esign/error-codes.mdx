## Error codes and messages for Aadhaar eSign APIs

Below is a table which contains the error codes and corresponding error messages, for errors that may occur during the eSign flow.

<table>
    <tbody>
        <tr>
            <th> Error Code </th>
            <th> Error Message </th>
        </tr>
        <tr>
            <td> session_expired </td>
            <td> The session has expired. Please try again. </td>
        </tr>
        <tr>
            <td> document_already_signed </td>
            <td> The document has already been signed successfully. </td>
        </tr>
        <tr>
            <td> document_not_found </td>
            <td> This document is not available anymore. </td>
        </tr>
        <tr>
            <td> signature_creation_failure </td>
            <td> Failed to generate signature. </td>
        </tr>
        <tr>
            <td> exceeded_otp_retries </td>
            <td> Exceeded OTP failure attempts. </td>
        </tr>
        <tr>
            <td> otp_not_entered </td>
            <td> OTP was not entered. </td>
        </tr>
        <tr>
            <td> signer_declined </td>
            <td> Document signing was declined. </td>
        </tr>
        <tr>
            <td> otp_expired </td>
            <td> OTP expired. </td>
        </tr>
        <tr>
            <td> aadhaar_not_linked </td>
            <td> Aadhaar not linked with mobile. </td>
        </tr>
        <tr>
            <td> service_down </td>
            <td> ESP service down. </td>
        </tr>
        <tr>
            <td> name_match_error </td>
            <td> Your name does not match with the data received from Aadhaar. </td>
        </tr>
        <tr>
            <td> yob_match_error </td>
            <td> Your year of birth does not match with the data received from Aadhaar. </td>
        </tr>
    </tbody>
</table>
