## Overview

The Account Availability Check API enables Financial Information Users (FIUs) to verify the existence of customer accounts across Account Aggregators Setuâ€™s AA gateway offers using a customer's phone number. This documentation provides guidelines on how to integrate and use the API effectively.

<hr class="primary" />

### Authentication

FIUs must use [Auth Mechanism](/data/account-aggregator/api-reference#/operation~getToken) to obtain an access token for authentication. Include the access token in the Authorization header of each request.

<hr class="primary" />

### Account Availability API

<CodeBlockWithCopy language="json">
    {`POST /v2/account-availability
Headers: 
    Authorization: Bearer <access token>
    Content-Type: application/json
    x-product-instance-id: <product instance id>
Body:
{
    "mobileNumber": "9999999999",
}`}
</CodeBlockWithCopy>
