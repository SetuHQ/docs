## Name Match API Overview

The Name Match API, when utilised in KYC verification, ensures accurate identity verification, detects errors and variations in names, and prevents identity fraud. 
It helps financial institutions comply with regulatory requirements like AML, enhances customer trust, and improves the efficiency and reliability of the verification process.

### What is Setu Name Match API?

The Setu Name Match API utilizes name-matching algorithms to accurately recognize variations of similar names from multiple sources. 
This allows organizations to make informed decisions on whether to accept or reject users during their onboarding process.
