## Test Insights APIs with Postman

#### Download and set up Postman

Download and setup the <a href="https://www.postman.com/downloads" target="_blank">Postman App</a>, if you haven’t already. All APIs required for using BSA have been published in this <a href="https://documenter.getpostman.com/view/15939114/2s93Jrw4mm" target="_blank">collection</a>. Click on **Run in Postman** button on the top right corner to import BSA’s API collection to Postman.

**Update collection variables**

<Callout type="tip">
    To edit collection variables in Postman, click on the collection and go to{" "}
    <strong>Variables</strong>.
</Callout>

<br />

Postman collection comes wth default test credentials. Set the following variable in the collection on Postman—

-   Set `client_id` to the FIU app client ID received on Bridge.
-   Set `client_secret` to the FIU app client secret received on Bridge.

![](https://storage.googleapis.com/strapi-assets/latest/Screenshot_2022-02-21_at_12.55.07_PM/Screenshot_2022-02-21_at_12.55.07_PM.png)

<Callout type="warning">
    Remove any new lines at the end after pasting values, to avoid errors.
</Callout>

<br />

Hit **Save** after all the changes are complete.
