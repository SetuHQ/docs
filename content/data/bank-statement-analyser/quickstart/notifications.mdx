## Notifications

The `base_url` is the same server URL configured for the FIU app to receive consent and data flow notifications.

<Callout type="tip">
    Please refer to the Account Aggregator's{" "}
    <a href="/data/account-aggregator/api-integration/notifications">
        notifications guide
    </a>{" "}
    for configuring API endpoints on your server to receive notifications from
    Setu Insights.
</Callout>

<br />

#### Insights notification

This notification is used to communicate the generated status of the insights, along with the generated insights.

###### Insights Notification Payload

Setu will post this payload to your notification endpoint when Insights are successfully generated.

<CodeBlockWithCopy language="json">
    {`{
  "type": "INSIGHT_STATUS_UPDATE",
  "timestamp": 2018-12-06T11:39:57.153Z,
  "jobId": "d4f40bd9-a22f-4408-a622-4e8a1e4fbda6",
  "success": true,
  "data": {
    "status": "READY", 
    "insights": {
      // ... insight_name : insight_value
    }
  },
  "error": {}
}`}
</CodeBlockWithCopy>
