FIU can create reports with the exact set of required Insights. Below is a list of all available Insights that the Setu BSA can provide,

| Name                                     | Title                                           |
| ---------------------------------------- | ----------------------------------------------- |
| `name`                                   | Name                                            |
| `dob`                                    | Date of Birth                                   |
| `email`                                  | E-Mail                                          |
| `pan`                                    | Permanent Account Number                        |
| `ckyc_compliance`                        | CKYC Compliance                                 |
| `mobile`                                 | Mobile Number                                   |
| `bank_names`                             | Bank Name                                       |
| `bank_branch`                            | Bank Branch                                     |
| `bank_account_number`                    | Bank Account Number                             |
| `bank_account_type`                      | Bank Account Type                               |
| `opening_date`                           | Account Opening Date                            |
| `closing_balance`                        | Account Closing Balance                         |
| `od_cc_limit`                            | OD/CC Limit                                     |
| `current_quarterly_average_eod_balance`  | Average End-of-day Balance for Current Quarter  |
| `previous_quarterly_average_eod_balance` | Average End-of-day Balance for Previous Quarter |
| `end_of_month_balances`                  | Balances at End-of-month                        |
| `balances_on_10th`                       | Balances at 10th of month                       |
| `end_of_day_balances`                    | End-of-day Balances                             |
| `average_end_of_day_balance`             | Average of End-of-day Balances                  |
| `maximum_end_of_day_balance`             | Maximum of End-of-day Balances                  |
| `total_credits`                          | Total number of Credits                         |
| `total_credits_value`                    | Total amount credited                           |
| `total_debits`                           | Total number of Debits                          |
| `total_debits_value`                     | Total amount debited                            |
| `top_10_credits`                         | Top 10 Credits                                  |
| `top_10_debits`                          | Top 10 Debits                                   |
| `quarterly_salary_average`               | 3-months Salary Average                         |
| `semi_yearly_salary_average`             | 6-months Salary Average                         |
| `yearly_salary_average`                  | 12-months Salary Average                        |
| `cash_deposit_x_total_credit`            | Monthly Cash Deposit to Total Credit Ratios     |
| `cash_withdrawals`                       | 10 Recent Cash Withdrawals                      |
| `inward_cheque_bounce_count`             | Total Number of Inward Cheque Bounces           |
| `outward_cheque_bounce_count`            | Total Number of Outward Cheque Bounces          |
| `cheque_bounces`                         | Summary of Cheque Bounces                       |
| `expenses_monthly_summary`               | Expense Classification: Monthly Summary         |
| `inflows_monthly_summary`                | Inflow Classification: Monthly Summary          |
